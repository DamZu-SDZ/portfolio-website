Technical Specifications

The system utilizes an **ESP32** microcontroller as the central processing unit. Authentication is handled via the **GM861 QR scanner** and **PN532 NFC module**, ensuring multi-layered access.

  • Real-time database communication via **XAMPP/MySQL**.
  • Automated solenoid lock triggering upon successful code verification.
  • Web-based interface for remote monitoring and log history.
