# 🌐 damzu.site — Personal Portfolio

Selamat datang ke repositori rasmi untuk **damzu.site**, hub portfolio peribadi saya yang memaparkan perjalanan akademik, projek kejuruteraan perisian, serta inovasi dalam bidang IoT (Internet of Things) dan Sistem Terbenam (Embedded Systems).

Laman web ini berfungsi sebagai resume digital interaktif dan ruang pameran bagi semua hasil kerja serta kepakaran teknikal saya.

---

## 🚀 Ciri-Ciri Utama (Features)

* **Reka Bentuk Responsif & Minimalis:** Dioptimumkan sepenuhnya untuk pengalaman pengguna yang lancar sama ada di peranti mudah alih, tablet atau desktop.
* **Showcase Projek Interaktif:** Bahagian khas yang memaparkan butiran projek utama, lengkap dengan timbunan teknologi (tech stack) dan pautan kod sumber.
* **Arkib Pengalaman & Pencapaian:** Ringkasan perjalanan akademik (termasuk latar belakang Kolej Matrikulasi Johor dengan CGPA 4.0), peranan kepimpinan, dan penglibatan komuniti.
* **Borang Hubungan Langsung:** Memudahkan bakal majikan atau rakan kolaborator untuk menghantar mesej atau berhubung terus melalui e-mel.

---

## 🛠️ Timbunan Teknologi (Tech Stack)

Laman web ini dibina menggunakan teknologi moden untuk memastikan prestasi yang pantas, keselamatan yang tinggi, dan kebolehskalaan (*scalability*):

* **Frontend:** Next.js (React Framework) & Tailwind CSS (Reka bentuk UI)
* **Backend & Autentikasi:** Firebase (Firestore / Authentication)
* **Pangkalan Data (Database):** Firebase Realtime Database / Firestore
* **Gubahan & Pengehosan (Hosting):** Vercel / Firebase Hosting
* **Domain:** Diuruskan melalui Namecheap (`damzu.site`)
